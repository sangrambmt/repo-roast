A brutally honest Python CLI that roasts your repository like a senior engineer who has seen too much.

Repo Roast scans your project and calls out the obvious problems developers usually ignore: weak README files, missing tests, giant files, missing licenses, bad setup hygiene, and messy structure.

It is part code quality tool, part comedy weapon.

---

## Why?

Because most repos do not need another dashboard.

They need someone to say:

> Your README explains the product but not how to run it. That is marketing, not documentation.

Repo Roast gives you fast, blunt feedback before other developers silently judge your project and leave.
